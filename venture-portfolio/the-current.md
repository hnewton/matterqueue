## Idea Capsule — 2026-06-16

**Name** — The Current

**One Phrase** — [FILL: one phrase — the sharpest distillation]

**Core Sentence** — A personal attention layer with frictionless capture and schema on read — everything you pay attention to, connected, queryable, and renderable as whatever output serves you in the moment

**One Paragraph** — Everything you pay attention to — notes, links, conversations, images — disappears into a dozen apps that don't talk to each other. The Current is a personal attention layer that captures all of it with zero friction, then lets you render it as whatever you need in the moment: a moodboard, a timeline, a brief, a knowledge graph, a recommendation. Your data lives in your own store, never ours, so sovereignty is the foundation rather than an afterthought. It's the most developed idea in the ecosystem, and every tool that feeds it makes the next one stickier.

**Key Insights**
- Attention is the signal — what you capture, return to, and connect is a more honest representation of who you are than any profile or resume
- Frictionless capture is the product — if capture has friction the system doesn't get fed; entire value depends on zero-resistance input across every context
- Schema on read not schema on write — data captured in natural form; structure applied at moment of use depending on output needed; moodboard, knowledge graph, timeline, brief, recommendation from same corpus
- User-owned data architecture — Supabase per-user project model; The Current never holds your data; sovereignty is the trust foundation
- PLG destination — tools feed it, it makes every tool stickier; it matures in parallel with PLG surfaces
